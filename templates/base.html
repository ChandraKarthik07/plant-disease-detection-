Features

This web app uses Deep Learning Algorithms to predict the diseases in your crops. It implements various deep learning libraries to build Robust and Efficient Neural Network which can provide you Fast and Accurate Results.

Technology Stack of this project:

  • python
  • HTML, CSS, JavaScript
  • Flask
  • TensorFlow, Keras
  • Pandas, Numpy, Scipy
